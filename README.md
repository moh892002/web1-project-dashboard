# StudyBuddy - Study Management App

## Project Description
StudyBuddy is a single-page web application (SPA) that helps students organize their study life. The app provides tools for managing daily tasks, tracking positive habits, and saving favorite educational resources.

## Features
- **Dashboard**: View a summary of tasks and completion percentage with:
  - Quick Add Task (Title + Due Date)
  - Tasks due soon (today or within 2 days)
  - Number of completed tasks
  - Progress bar showing completion percentage
  - Today's habits streak
- **Tasks**: Add, delete, and mark tasks as completed with priority and category selection
- **Habits**: Track daily habits with a weekly goal
- **Resources**: Save and search educational resources
- **Settings**: Switch theme (light/dark) and reset data
- **Local Storage**: All data is automatically saved in the browser

## How to Run
1. Download the files to a folder on your device
2. Open the `index.html` file in any web browser
